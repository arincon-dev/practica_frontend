# Frontend Testing

Frontend test stack:
- Jasmine
- Karma
- Angular TestBed

Recommended test coverage:
- Component rendering and interaction
- Service HTTP calls with mocked API responses
- Form validation and popup mode behavior

Current spec files:
- `src/app/features/login/login.component.spec.ts`
- `src/app/features/user-list/user-list.component.spec.ts`
- `src/app/features/user-popup/user-popup.component.spec.ts`

Run tests:

```bash
npm test
```
