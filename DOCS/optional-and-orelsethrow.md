# Nullability and Defensive Access

TypeScript equivalent of Java Optional patterns:
- Optional chaining (`obj?.prop`)
- Nullish coalescing (`value ?? fallback`)
- Early returns for missing required state

Use cases in project:
- Session credentials retrieval
- Optional fields from backend response
- Popup update guards when IDs are missing
